pydarkstar¶

A set of command-line tools for interacting with a Darkstar server.

  • The pydarkstar website can be found here.

  • The Setup instructions can be found here (start here!)

  • The Usage instructions can be found here.

Please note that this project was developed for the original darkstar server. That project was archived and forked a few time. This project may work with the forks if the AH implementation remains unchanged.

Features¶

  • Create Auction House item database with prices.

  • Detailed control over the buying and selling of items.

  • Buy items put up for sale by players.

  • Stock the Auction House with items for players to buy.
